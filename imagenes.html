Apuntes

Ayudan a los buscadores a indexar las páginas y brindan una buena experiencia de usuario (UX), además de proporcionar un descanso en la lectura al usuario.

El atributo src es dónde se ubica la imagen a utilizar, se puede poner directamente de la web o descargada en el PC.

En caso de tenerla en el PC, se tiene que especificar la ruta de la imagen, si esta se encuentra dentro de una carpeta, se escribe primero el nombre de la carpeta y luego el nombre de la imagen. Los atributos width y height sirven para modificar el tamaño de la imagen, este siempre es proporcional, nunca se deberán usar ambos.

El atributo alt sirve para mostrar un texto alternativo en caso de que la imagen no se encuentre. Este también es importante para el buscador puesto que este texto es la información que utiliza para esa imagen y esta mostrará tu página.

Una buena práctica en el uso de imágenes, es el reducir el tamaño de las mismas, ya que al momento de reducir el tamaño, este solo es visual, por lo que se debe usar Photoshop para reducir su tamaño en kb

Resumen

Se explicaron las 2 formas de agregar imágenes, sus atributos src, alt, width, height, además de una buena práctica para las mismas
