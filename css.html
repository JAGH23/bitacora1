Apuntes

Con el CSS se pueden agregar fuentes, colores, márgenes, tipos de letra, animaciones, etc. Hay 3 maneras de implementar CSS, de manera incrustada, estilo en linea y externa.

De manera incrustada se hace con la etiqueta "style" y se coloca en el head, puede usarse en un documento único, pero para múltiples es mejor usar un CSS externo. De manera en línea puede ser aplicado a cualquier elemento HTML (etiquetas) incluyendo el body. De manera externa se utiliza la etiqueta link, la cual enlaza el documento externo de CSS, pero se debe especificar la ruta donde se encuentra, al usar el link, el atributo "rel" indica que es hoja de estilo, el "type" es opcional, dentro del href es la ruta donde está guardado y cómo se llama.

Su sintaxis se conforma por el selector, la declaración, la propiedad y su valor. Ejemplo: #principal{font-family:arial;}

El orden siempre es en cascada, por lo que hay una prioridad si se eligen utilizar diferentes maneras de implementar el CSS, uno tendrá mas prioridad que el otro. El orden es el siguiente: Estilo en línea, hoja de estilo externa e interna y el predeterminado del navegador.

El box model se utiliza para visualizar qué espacio ocupan los elementos. Border y padding hacen crecer a los elementos. Es recomendable solo utilizar el módulo Flex box en la barra de navegación, al usar display flex se activan más de 10 propiedades.
