Estructura básica

Apuntes

HTML significa Hypertext Markup Language, es el componente mas básico de la web, define la estructura y el significado del contenido web.

La declaración del HTML no tiene cierre.

Utiliza antilambdas o diples para etiquetar texto, imagenes y otro contenido. La mayoría de las etiquetas tienen inicio y fin.

Sigue una jerarquía, la etiqueta principal se le llama padre, los que le siguen se llaman hermanos. La página principal siempre se llama index.
