Apuntes

Un enlace puede ser cualquier cosa, texto, imagen, video, botón, etc. Dentro de la sintaxis se encuentra la apertura de etiqueta "a", href define el destino del enlace, lo que se encuentra entre comillas es el destino o url, el texto es la parte visible y finalmente el cierre.

Un enlace tiene diferentes estados, el no visitado esta subrayado en azul, el visitado está subrayado en morado y el activo está subrayado y en rojo. Esto se puede cambiar con CSS. Para poner un link en la imagen, dentro de la etiqueta "a" se pone la etiqueta img. El atributo target especifica dónde abrir el documento vinculado y puede tener los valores _self (predeterminado), _blank (se abre una nueva ventana o pestaña), _parent (lo abre en el marco primario), _top (cuerpo completo de la ventana).

Las anclas sirven para ir a partes específicas de la página, se usa la dirección donde se encuentra la página y se agrega la id de la sección.
